============================================================
1. 와우아이디어스 소개
============================================================

[정의]
와우아이디어스는 브레인스토밍 기반 과정 중심 PBL(Project-Based Learning) 협업 플랫폼입니다. 결과뿐 아니라 아이디어가 발전하는 모든 과정을 담고 협업을 지원하는 것을 목표로 합니다.

[주요 특징]
- 창의적 아이디어 발상, 상호작용, 평가, 공유가 하나의 플랫폼에서 이루어지는 원스톱 솔루션
- 2023년 교육부 장관상 수상으로 교육 효과성 인정
- 별도의 앱 설치 없이 PC, 스마트폰, 태블릿 등 다양한 디바이스에서 바로 사용 가능한 반응형 웹
- 크롬북에서도 사용 가능하여 교육 현장에서의 접근성이 높음

[요금 정책]
회원가입 후 3개의 프로젝트(총 30개 작업장)를 기간 제한 없이 무료로 사용할 수 있어 기본 기능 경험에 충분합니다. 더 많은 프로젝트가 필요할 경우 유료로 전환 가능합니다.


============================================================
2. 시작하기 전 필수 준비사항
============================================================

[웹 브라우저 환경]
이 매뉴얼은 크롬(Chrome) 브라우저와 센스리더를 기준으로 작성되었습니다.

[가상커서 설정 확인]
웹 기반 플랫폼 사용 시 센스리더의 가상커서 설정이 매우 중요합니다.

- 가상커서 사용 선택: Ctrl+Shift+F11
- 가상커서 사용 해제: Ctrl+Shift+F12

와우아이디어스에서 단축키를 사용하거나 입력창에 텍스트를 입력할 때는 반드시 가상커서를 해제해야 합니다. 입력이 되지 않거나 단축키가 작동하지 않으면 Ctrl+Shift+F12를 눌러 가상커서를 해제한 후 다시 시도하세요.

[탐색 모드 전환]
- 기능키 모드: 웹페이지를 탐색할 때 사용 (ESC 키로 전환)
- 입력 모드: 편집창에 텍스트를 입력할 때 사용 (Space Bar 또는 Tab 키로 전환)

편집창에서 입력이 안 되면 Space Bar를 눌러 입력 모드로 전환하세요.


============================================================
3. 와우아이디어스 접속 및 로그인
============================================================

[접속 방법]
1. 크롬 브라우저를 실행합니다.
2. 주소창(Alt+D)에 다음 주소를 입력합니다: wowstorming.com
3. Enter를 누릅니다.

[로그인 방법]
1. 메인 페이지에 접속하면 Tab 키를 눌러 '로그인' 버튼을 찾습니다.
2. 로그인 버튼에서 Enter를 누릅니다.
3. 이메일 주소 입력창에 포커스가 이동합니다. 가입된 이메일 주소를 입력합니다.
4. Tab 키를 눌러 비밀번호 입력창으로 이동한 후 비밀번호를 입력합니다.
5. Enter를 누르거나 Tab 키로 '로그인' 버튼을 찾아 Enter를 누릅니다.

[스크린리더 사용자 팁]
로그인 후 메인 화면에서는 Tab 키를 눌러 주요 메뉴와 버튼을 탐색할 수 있습니다. 센스리더는 각 요소의 역할(버튼, 링크, 편집창 등)을 음성으로 안내해 줍니다.


============================================================
4. 기본 구조 이해하기
============================================================

[3단계 위계 구조]
와우아이디어스는 다음과 같은 3단계 구조를 가지고 있습니다.

1단계: 프로젝트(Project) - 가장 상위 단위
2단계: 작업장(Workspace) - 프로젝트 안에 여러 개 생성 가능
3단계: 아이디어(Idea) - 작업장 안에 개별 아이디어 등록

[구조 이해의 중요성]
이 3단계 구조를 이해하면 전체 탐색이 용이합니다. 다른 협업 툴이 '보드' 단위인 것과 달리, 와우아이디어스는 여러 작업장(보드)을 묶는 '프로젝트'가 기본 단위입니다.

[스크린리더 사용자 관점]
- Tab 키로 탐색 시 "프로젝트 제목", "작업장 제목", "아이디어 카드" 등으로 계층이 구분되어 읽힙니다.
- 계층 구조를 파악하려면 센스리더의 헤딩 이동 기능(H 키)을 활용하세요.


============================================================
5. 수업 전 준비 단계
============================================================

[5-1. 프로젝트 생성하기]

[개념]
새 수업 단위를 '프로젝트'로 생성합니다. 하나의 프로젝트는 한 학기 수업 또는 특정 주제의 프로젝트 학습 단위로 설정할 수 있습니다.

[실행 방법]
1. 로그인 후 메인 화면에서 Tab 키를 눌러 '새 프로젝트 만들기' 버튼을 찾습니다.
2. Enter를 누르면 프로젝트 생성 화면으로 이동합니다.
3. 프로젝트 제목 입력창에 수업명이나 주제를 입력합니다.
   예: "2학년 1학기 창의적 문제해결 수업"
4. Tab 키로 '생성' 또는 '확인' 버튼을 찾아 Enter를 누릅니다.

[프로젝트 복사 기능]
교과 전담 교사처럼 여러 반에 동일한 수업을 진행할 경우, 하나의 프로젝트를 완성한 뒤 '프로젝트 복사' 기능을 사용하면 반복 작업을 획기적으로 줄일 수 있습니다. 한 반의 모둠 구성, 차시별 지시문, 예시 자료까지 모두 포함된 프로젝트 전체가 복사되어 매우 편리합니다.


[5-2. 작업장(Workspace) 구성하기]

[개념]
프로젝트 내에서 수업 계획에 따라 여러 개의 '작업장'을 생성할 수 있습니다. 작업장은 차시별 활동이나 단계별 과제로 구성할 수 있으며, 작업장 아래에 소작업장을 만드는 계층 구조화가 가능하여 체계적인 수업 설계가 가능합니다.

[UI 스타일 선택]
작업장 생성 시 '카드', '캔버스', '칸반', '상관망' 등 다양한 스타일의 UI를 선택할 수 있습니다. 이는 특정 발상 기법을 효과적으로 적용하기 위함입니다.

- 칸반 스타일 UI: 스캠퍼(SCAMPER)나 PMI 기법에 적합
- 카드 스타일: 브레인스토밍에 적합
- 캔버스 스타일: 자유로운 배치와 연결이 필요한 활동에 적합

[실행 방법]
1. 프로젝트 화면에서 Tab 키로 '작업장 추가' 또는 '새 작업장' 버튼을 찾습니다.
2. Enter를 누르면 작업장 생성 옵션이 나타납니다.
3. 작업장 제목을 입력합니다.
   예: "1차시 브레인스토밍", "2차시 아이디어 분류"
4. Tab 키로 UI 스타일 선택 옵션을 찾아 원하는 스타일을 선택합니다.
5. '생성' 버튼에서 Enter를 누릅니다.


[5-3. 학생 관리하기]

[학생 일괄 등록]
학생 관리 도구를 통해 학생을 일괄 등록할 수 있으며, 모든 라이선스에 기본으로 제공되는 기능입니다.

[비밀번호 초기화]
학생이 비밀번호를 잊었을 때 교사(관리자)가 직접 '비밀번호 초기화'를 해줄 수 있어 즉각적인 대응이 가능합니다.

[사고 성향 검사 기반 소그룹 구성]
'사고 성향 검사' 도구를 통해 학생들의 사고 성향을 파악하고, 서로 다른 성향의 학생들로 소그룹을 자동 생성해주는 기능을 제공합니다. 네드 허먼의 '전체 대뇌 모형'을 기반으로 하며, 시너지를 극대화하는 모둠 구성에 유용합니다.

[스크린리더 사용자 팁]
학생 관리 화면에서는 표(테이블) 형식으로 학생 목록이 표시됩니다. 센스리더의 테이블 탐색 기능을 활용하세요.
- 테이블 행 이동: Ctrl+Alt+위/아래 방향키
- 테이블 열 이동: Ctrl+Alt+왼쪽/오른쪽 방향키

6. 수업 중 활동 단계
============================================================

[6-1. 아이디어 발상하기]

[다양한 매체 활용]
글, 그림, 스케치, 인터넷 자료(링크), 유튜브 영상 등 다양한 매체를 사용하여 아이디어를 실시간으로 업로드할 수 있습니다. 온라인 환경에서 포스트잇 없이 다양한 발상 활동이 가능합니다.

[아이디어 등록 방법]
1. 작업장 화면에서 Tab 키로 '아이디어 추가' 또는 '+' 버튼을 찾습니다.
2. Enter를 누르면 아이디어 입력창이 나타납니다.
3. 가상커서를 해제(Ctrl+Shift+F12)하고 텍스트를 입력합니다.
4. 입력이 완료되면 Tab 키로 '저장' 또는 '등록' 버튼을 찾아 Enter를 누릅니다.

[링크나 파일 첨부]
아이디어 입력창에서 Tab 키를 눌러 '파일 첨부' 또는 '링크 추가' 버튼을 찾을 수 있습니다.


[6-2. 발상 촉진 기능 활용하기]

[타이머 기능]
제한 시간 내 집중을 유도합니다.

[Hidden-Show 기능]
다른 학생의 아이디어를 보지 못하게 하여 아이디어 '커닝'을 방지하는 기능입니다. 브레인스토밍의 '양 추구' 원칙을 온라인에서 효과적으로 구현하도록 돕습니다.

[익명/실명 전환]
선입견 없는 아이디어 제시나 프라이버시 보호가 필요할 때 사용합니다.


[6-3. 상호 작용하기]

[댓글 기능]
실시간으로 피드백을 제공할 수 있습니다. 교사는 '관리자 댓글'로 구분하여 표시할 수 있습니다.

[댓글 작성 방법]
1. Tab 키로 아이디어 카드를 탐색하다가 원하는 아이디어에서 Enter를 누릅니다.
2. 아이디어 상세 화면에서 Tab 키로 '댓글 입력' 또는 '피드백 작성' 편집창을 찾습니다.
3. 가상커서를 해제하고 댓글을 입력합니다.
4. Tab 키로 '댓글 등록' 버튼을 찾아 Enter를 누릅니다.

[별표 평가 기능]
직관적인 별표 기능으로 동료 평가를 진행하고 우수 아이디어를 식별할 수 있습니다.

[기타 기능]
- 히치하이킹: 타인의 아이디어를 개선하여 새 아이디어로 등록
- 쪽지: 관리자가 학생에게 1:1 메시지 전송


============================================================
7. 수업 후 평가 및 관리
============================================================

[7-1. 정량적 통계 데이터 확인하기]

[통계 데이터 메뉴]
'통계 데이터' 메뉴에서 학생별 접속 횟수, 아이디어 제출 수, 댓글 수, 받은 별표 개수 등을 수치로 확인할 수 있습니다. 이를 통해 참여도를 객관적으로 파악 가능합니다.

프로젝트 전체 통계뿐 아니라 각 차시(작업장)별 통계도 제공되어 특정 활동의 참여도를 세밀하게 분석할 수 있습니다.

[접근 방법]
1. 프로젝트 메인 화면에서 Tab 키로 '통계' 또는 '데이터 분석' 메뉴를 찾습니다.
2. Enter를 눌러 통계 화면으로 이동합니다.
3. 센스리더의 테이블 탐색 기능(Ctrl+Alt+방향키)을 사용하여 학생별 데이터를 확인합니다.


[7-2. 자동 생성 학생 포트폴리오]

[개념]
가장 강력한 기능 중 하나로, 학생별로 모든 활동 내용을 자동으로 모아 포트폴리오를 생성해 줍니다. 학생이 각 작업장에서 어떤 아이디어를 냈는지 시간 순으로 추적 가능합니다.

학생의 사고 변화와 성장 과정을 관찰하는 데 매우 유용하며, 과정 중심 평가의 핵심 자료로 활용할 수 있습니다. 최근 업데이트로 사고 성향과 정량적 통계까지 함께 제공됩니다.

[접근 방법]
1. 학생 관리 또는 통계 메뉴에서 Tab 키로 특정 학생 이름을 찾습니다.
2. Enter를 눌러 학생 상세 정보로 이동합니다.
3. Tab 키로 '포트폴리오 보기' 또는 '활동 이력' 버튼을 찾아 Enter를 누릅니다.


[7-3. 프레젠테이션 기능]

[개념]
와우아이디어스 플랫폼 내에서 제출된 아이디어들을 선택하고 순서를 재배치하여 간단하게 발표 자료를 만들 수 있습니다. 다른 프로그램으로 이동할 필요 없이 결과물을 바로 공유하고 발표할 수 있습니다.

[실행 방법]
1. 작업장 화면에서 Tab 키로 '프레젠테이션' 또는 '발표 모드' 버튼을 찾습니다.
2. Enter를 누르면 아이디어 선택 화면으로 이동합니다.
3. 발표에 포함할 아이디어를 선택하고 순서를 조정합니다.
4. '프레젠테이션 시작' 버튼을 눌러 발표를 진행합니다.

[부록] 스크린리더 주요 단축키 정리
============================================================

[센스리더 기본 단축키]
- 가상커서 사용 선택: Ctrl+Shift+F11
- 가상커서 사용 해제: Ctrl+Shift+F12
- 기능키 모드로 전환: ESC
- 입력 모드로 전환: Space Bar

[웹 탐색 단축키]
- 다음 헤딩으로 이동: H
- 이전 헤딩으로 이동: Shift+H
- 다음 링크로 이동: L
- 이전 링크로 이동: Shift+L
- 다음 편집창으로 이동: F2
- 이전 편집창으로 이동: Shift+F2
- 다음 버튼으로 이동: B (헤딩과 다름, 실제로는 Tab 권장)
- 다음 폼 컨트롤로 이동: N
- 이전 폼 컨트롤로 이동: Shift+N

[테이블 탐색 단축키]
- 테이블 다음 행으로 이동: Ctrl+Alt+아래 방향키
- 테이블 이전 행으로 이동: Ctrl+Alt+위쪽 방향키
- 테이블 다음 열로 이동: Ctrl+Alt+오른쪽 방향키
- 테이블 이전 열로 이동: Ctrl+Alt+왼쪽 방향키
- 표 정보 읽기: Ctrl+Alt+F9

[내용 읽기 단축키]
- 현재 줄 읽기: Ctrl+Shift+I
- 현재 단어 읽기: Ctrl+Shift+K
- 연속 읽기: F11
- 연속 읽기 중지: ESC
