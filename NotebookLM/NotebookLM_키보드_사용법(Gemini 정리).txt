# Google NotebookLM 키보드 사용법 및 센스리더 활용 가이드

이 문서는 '김헌용_대전맹학교_디지털연수1(Clovanote 녹취록).txt' 파일의 내용을 바탕으로, 시각장애인 사용자가 키보드와 센스리더 스크린리더를 사용하여 Google NotebookLM을 탐색하고 작동하는 방법을 정리한 것입니다.

## 1. 기본 탐색 (센스리더 단축키)

웹 브라우저(Chrome, Edge 등)에서 NotebookLM 사이트를 탐색할 때 유용한 주요 센스리더 단축키입니다.

-   **헤딩 간 이동**: `H` (다음 헤딩), `Shift+H` (이전 헤딩)
-   **헤딩 레벨별 이동**: 숫자 키 `1`~`6` (예: `2`를 누르면 2레벨 헤딩 간 이동)
-   **버튼 간 이동**: `N`
-   **편집창 간 이동**: `F2`
-   **가상 커서 켜기**: `Ctrl+Shift+F11`
-   **가상 커서 끄기**: `Ctrl+Shift+F12`. (오디오 플레이어 등 일부 기능을 조작하려면 가상 커서를 꺼야 합니다.)

## 2. NotebookLM 시작 및 노트 생성

1.  **사이트 접속**: 웹 브라우저 주소창에 `notebooklm.google.com`을 입력합니다.
2.  **로그인**: 구글 계정으로 로그인합니다.
3.  **새 노트 만들기**: `N`키로 '새 노트 만들기' 버튼을 찾아 `Enter`를 누릅니다.
4.  **소스 추가**:
    -   '소스 업로드' 화면이 나타나면, 방향키나 `F2`키를 사용하여 '소스 검색'으로 이동 후 `Enter`를 누릅니다. (컴퓨터의 파일을 직접 업로드할 수도 있습니다.)
    -   '소스 탐색에 사용할 쿼리' 편집창이 나타나면, 학습하고 싶은 주제의 키워드를 입력합니다.
    -   `Tab`키나 방향키로 '쿼리 제출' 버튼으로 이동하여 `Enter`를 누릅니다.
    -   검색된 10개의 소스가 기본적으로 선택됩니다. 페이지 하단으로 이동하여 '가져오기' 버튼을 찾아 `Enter`를 누릅니다.

## 3. 화면 구성 및 탭 간 이동

노트가 생성되면 화면은 크게 3개의 탭으로 구성됩니다. 각 탭은 **헤딩 레벨 2**이므로 숫자 키 `2`로 쉽게 이동할 수 있습니다.

-   **출처 (Sources)**: 왼쪽에 위치하며, 추가한 소스 목록을 보여줍니다.
-   **채팅 (Chat)**: 중앙에 위치하며, AI와 대화하는 공간입니다.
-   **스튜디오 (Studio)**: 오른쪽에 위치하며, 소스를 기반으로 오디오, 보고서 등 다양한 콘텐츠를 생성합니다.

*팁: 만약 3개의 탭이 모두 보이지 않고 '채팅'만 보인다면, `Windows 키 + 위쪽 화살표`를 눌러 브라우저 창을 최대화하세요.*

## 4. 채팅 탭 활용하기

1.  **채팅창 이동**: `H`키나 숫자 `2`키로 '채팅' 헤딩으로 이동한 후, `F2`키를 두 번 눌러 질문을 입력하는 '쿼리 상자' 편집창으로 이동합니다.
2.  **질문하기**: 소스와 관련된 질문을 입력하고 `Enter`를 누릅니다.
3.  **답변 확인**: 답변은 편집창 아래에 생성됩니다.
4.  **대화 간 이동**: 각 질문과 답변은 **헤딩 레벨 3**으로 구성됩니다. 숫자 키 `3`을 사용하여 이전/다음 대화로 빠르게 이동할 수 있습니다.
5.  **답변 복사**: 각 답변 아래에는 '클립보드에 모델 대답 복사' 버튼이 있어, `N`키로 찾아 `Enter`를 누르면 답변 내용을 복사할 수 있습니다.

## 5. 스튜디오 탭 활용하기 (콘텐츠 생성)

### AI 오디오 오버뷰 (팟캐스트) 생성

1.  **스튜디오로 이동**: `H`키나 숫자 `2`키로 '스튜디오' 헤딩으로 이동합니다.
2.  **생성 버튼 선택**: 'AI 오디오 오버뷰'라는 텍스트 바로 위에 있는 '내 소스를 기반으로 AI 팟캐스트를 생성합니다' 버튼을 찾아 `Enter`를 누릅니다.
3.  **옵션 설정**:
    -   **형식**: '심층 분석', '요약', '비평', '토론' 등 원하는 스타일을 선택합니다. (`Enter`를 눌러도 선택되었다는 안내는 나오지 않지만 적용됩니다.)
    -   **길이**: '짧게' 또는 '기본'을 선택합니다.
4.  **생성 시작**: '생성' 버튼을 찾아 `Enter`를 누릅니다. 생성이 완료되기까지 5~10분 정도 소요될 수 있습니다.

### 보고서 생성

1.  **스튜디오로 이동** 후 '보고서 생성' 버튼을 찾아 `Enter`를 누릅니다.
2.  **형식 선택**: '브리핑 문서', '학습 가이드', '보도 자료' 등 원하는 보고서 형식을 선택하고 `Enter`를 누릅니다.
3.  **생성 시작**: '생성' 버튼을 한 번 더 눌러 보고서 생성을 시작합니다.

## 6. 생성된 콘텐츠 확인 및 조작

**※ 중요: 오디오 플레이어나 보고서 등 생성된 콘텐츠를 조작하려면 반드시 가상 커서를 꺼야 합니다.**

### 오디오 플레이어 조작

1.  생성된 오디오는 페이지 하단에 플레이어 형태로 나타납니다.
2.  **가상 커서를 끈 상태**에서 `Tab`키와 `Shift+Tab`키로 플레이어 내부를 이동합니다.
3.  **주요 컨트롤**:
    -   **재생/일시정지**: '오디오 재생' 또는 '일시정지' 버튼에서 `Space`나 `Enter`를 누릅니다.
    -   **추가 옵션**: '오디오 플레이어 옵션 더 보기' 버튼에서 `Enter`를 누르면 메뉴가 열립니다.
    -   **재생 속도 변경**: 방향키로 0.5배부터 2배까지 조절할 수 있습니다.
    -   **다운로드**: MP3 파일로 다운로드할 수 있습니다.
    -   **닫기**: '오디오 플레이어 닫기' 버튼으로 플레이어를 닫을 수 있습니다.

### 보고서 확인

1.  생성된 보고서는 페이지 하단에 버튼 형태로 나타납니다.
2.  버튼에 포커스가 있을 때 `Enter`를 쳐도 내용이 열리지 않으면, **가상 커서를 끈 상태**에서 해당 버튼을 다시 찾아 `Enter`를 누릅니다.
3.  보고서 뷰어가 열리면 '복사' 버튼을 이용해 내용을 복사할 수 있습니다.

## 7. 참고: 브라우저 내장 TTS 기능

-   **Microsoft Edge**: 페이지에 접속한 후 `Ctrl+Shift+U`를 누르면 내장된 TTS가 페이지를 읽어줍니다. (센스리더 기능 키 무시 모드에서 작동할 수 있습니다.)
-   **Chrome**: 'Read Aloud' (`Alt+P`), 'Speechify' (`Alt+A`) 같은 확장 프로그램을 설치하면 유사한 기능을 사용할 수 있습니다.
