1. 센스리더 기본 탐색 키
========================================

[편집창 이동]
- F2: 편집창 간 이동
  * 첫 번째 편집창: 노트 제목
  * 두 번째 편집창: 채팅 쿼리 상자

[버튼 간 이동]
- N: 버튼 간 이동 (단일 키 탐색)

[헤딩 이동]
- H: 다음 헤딩으로 이동
- Shift+H: 이전 헤딩으로 이동
- 숫자 1: 헤딩 1 레벨만 이동 (가장 큰 제목, 노트 제목)
- 숫자 2: 헤딩 2 레벨만 이동 (출처, 채팅, 스튜디오 탭 간 이동)
- 숫자 3: 헤딩 3 레벨만 이동 (채팅 질문 간 이동)

[방향키 탐색]
- 위/아래 방향키: 요소 간 탐색
- 왼쪽/오른쪽 방향키: 풀다운 메뉴에서 옵션 선택


========================================
2. NotebookLM 화면 구조 탐색
========================================

[3개 탭 구조]
NotebookLM은 3개의 주요 탭으로 구성됩니다:
1) 출처 (왼쪽 탭)
2) 채팅 (가운데 탭)
3) 스튜디오 (오른쪽 탭)

[탭 간 이동 방법]
- H 또는 숫자 2: 헤딩 2 레벨로 탭 간 이동
- 창 최대화 필요: Windows 키 + 위 화살표

[헤딩 레벨별 의미]
- 헤딩 1: 노트 제목 (가장 큰 제목)
- 헤딩 2: 출처, 채팅, 스튜디오 (탭 구분)
- 헤딩 3: 채팅 질문 제목 (질문 간 이동)


========================================
3. 페이지 이동 단축키
========================================

[페이지 단위 이동]
- Ctrl+Home: 화면 최상단으로 이동
- Ctrl+End: 페이지 최하단으로 이동

[탭 이동]
- Tab: 다음 요소로 이동
- Shift+Tab: 이전 요소로 이동

[창 관리]
- Windows 키 + 위 화살표: 창 최대화
- Esc: 취소, 메뉴 닫기


========================================
4. 소스 추가 및 노트 생성
========================================

[노트 만들기 순서]
1. NotebookLM 홈페이지에서 "새 노트 만들기" 버튼 찾기
2. Enter 또는 Space: 버튼 실행
3. 소스 업로드 창 열림
   - 방향키로 옵션 탐색:
     * 파일 선택
     * 링크
     * 웹사이트
     * 유튜브
     * 텍스트 붙여넣기
     * 소스 검색

[소스 검색으로 노트 만들기]
1. "소스 검색"으로 이동 → Enter
2. 멀티 라인 편집창에 검색 키워드 입력
3. "쿼리 제출" 버튼으로 이동 → Enter
   * 주의: 편집창에서 Enter를 치면 줄만 바뀜
4. 10개의 소스가 자동 선택됨 (확인만 하고 수정 불필요)
5. "가져오기" 버튼으로 이동 → Enter
6. 노트 생성 완료 (제목 자동 생성)

[노트 제목 확인 및 수정]
- Ctrl+Home으로 최상단 이동
- F2로 첫 번째 편집창 이동
- 제목 직접 수정 가능


========================================
5. 채팅 탭 사용법
========================================

[채팅 탭으로 이동]
- H (또는 숫자 2)로 "채팅" 헤딩 찾기
- 방향키로 아래로 내려가면:
  * 노트북 구성 개요
  * 소스 10개 설명
  * 메모에 저장
  * 요약 복사
  * AI 동영상 오버뷰
  * AI 오디오 오버뷰
  * 마인드맵
  * 쿼리 상자 (멀티 라인 편집창)

[질문하기]
1. F2로 쿼리 상자 편집창으로 이동
2. 질문 입력
3. Enter: 쿼리 제출
4. 방향키 아래로 내려가면 답변 확인

[답변 확인 및 관리]
- 답변마다 헤딩 3으로 표시됨
- 숫자 3: 질문 간 이동 (헤딩 3 레벨)
- 답변 밑에 버튼들 (N 키로 이동):
  * 메시지를 메모에 저장
  * 클립보드에 모델 대답 복사
  * 대답을 좋음으로 평가
  * 대답을 좋지 않음으로 평가

[NotebookLM 특징]
- 소스 10개 내에서만 정직하게 답변
- 환각 현상 극도로 적음
- 소스의 원천을 함께 표시


========================================
6. 스튜디오 탭 사용법
========================================

[스튜디오 탭으로 이동]
- H (또는 숫자 2)로 "스튜디오" 헤딩 찾기
- 방향키로 내려가면:
  1. AI 오디오 오버뷰 (팟캐스트 생성)
  2. AI 동영상 오버뷰
  3. 마인드맵
  4. 보고서
  5. 플래시 카드
  6. 대화용 퀴즈

[AI 오디오 오버뷰 생성]
1. "내 소스를 기반으로 AI 팟캐스트를 생성합니다" 버튼에서 Enter
2. 형식 선택:
   - 타일 그룹 / 심층 / 요약 / 비평 / 토론
   - Enter로 선택 (선택 확인은 안 됨)
3. 길이 선택:
   - 짧게 (6~7분) / 기본 (15분)
   - Enter로 선택
4. 언어 선택: 기본 한국어 (구글 계정 언어 기준)
5. (선택) AI 호스트 집중 사항 입력
6. "생성" 버튼에서 Enter
7. 생성 확인:
   - Ctrl+End로 페이지 최하단 이동
   - 방향키 위로 4~5번: "AI 오디오 오버뷰 생성 중" 확인
8. 생성 시간: 5~10분 소요

[보고서 생성]
1. "보고서 생성" 버튼에서 Enter
2. 형식 선택:
   - 기본: 브리핑 문서, 학습 가이드, 블로그 게시물
   - 추천: 주제에 따라 다름 (보도 자료, 사업 제안서 등)
3. 원하는 형식에서 Enter
4. "생성" 버튼에서 Enter
5. 생성 확인:
   - Ctrl+End로 최하단 이동
   - 방향키 위로: "보고서 생성 중" 확인


========================================
7. 오디오 플레이어 조작
========================================

[오디오 플레이어 열기]
- 오디오 오버뷰 생성 완료 후 자동 재생 시작
- Ctrl+End로 최하단 이동
- 방향키 위로: 설명이 길게 나오는 위치에서 Enter

[가상 커서 해제 필수]
- 오디오 플레이어 조작 전 반드시 가상 커서 해제
- Ctrl+Shift+F12

[플레이어 버튼 (Tab으로 이동)]
- 제목
- 메모 추가
- 좋은 토론
- 나쁜 토론
- 오디오 플레이어 옵션 더 보기
- 오디오 플레이어 닫기
- 오디오 재생 일시정지

[일시정지/재생]
- "오디오 재생 일시정지" 버튼에서 Enter 또는 Space

[옵션 더 보기 (메뉴)]
1. Shift+Tab으로 "오디오 플레이어 옵션 더 보기" 찾기
2. Enter로 메뉴 열기
3. 풀다운 메뉴 (오른쪽 화살표로 이동):
   - 재생 속도 변경: 0.5배 / 0.8배 / 1배 / 1.2배 / 1.5배 / 1.8배 / 2배
   - 다운로드: MP3로 다운로드 (다운로드 폴더에 저장)
   - 공유: 링크 공유

[재생 시간 슬라이드]
- Tab으로 "오디오 재생 시간 슬라이드" 찾기
- 방향키로 이동 잘 안 됨 (한계)
- Home 키: 오디오 재생 처음으로 이동

[플레이어 닫기]
- "오디오 플레이어 닫기" 버튼에서 Space


========================================
8. 보고서 생성 및 열기
========================================

[보고서 확인 위치]
- Ctrl+End로 페이지 최하단 이동
- 방향키 위로: 생성된 보고서 찾기
- 오디오 오버뷰 위에 보고서가 생성됨

[보고서 열기 (중요!)]
** 가상 커서를 반드시 해제한 상태에서 진행 **

1. 보고서 제목과 내용이 읽히는 위치 찾기
2. 가상 커서 해제 (Ctrl+Shift+F12)
3. Enter 
4. 스튜디오 탭 바로 아래로 보고서 뷰어 쭉 펼쳐짐
- "풀다운 더 보기"라고 읽으면: 가상 커서가 켜진 상태
  → 가상 커서 해제 후 다시 시도
- "더 보기" 버튼이 아닌 제목만 있는 버튼에서 Enter

[보고서 뷰어 조작]
- 제목 편집창: "아티팩트 제목"이라고 읽어주는 편집창에서 보고서 제목 수정 가능
- 복사 버튼: 보고서 내용 복사
- 삭제 버튼: 보고서 삭제
- 보고서 뷰어 닫기: 보고서 닫기


========================================
9. 가상 커서 제어
========================================

[가상 커서란?]
- 센스리더에서 웹 페이지를 탐색하는 모드
- 켜짐: 페이지 내용 읽기 및 탐색
- 꺼짐: 버튼 클릭, 입력 등 상호작용

[가상 커서 켜기/끄기]
- Ctrl+Shift+F11: 가상 커서 켜기
- F12: 가상 커서 끄기

[가상 커서를 꺼야 하는 경우]
1. 오디오 플레이어 조작
2. 보고서 열기
3. 생성된 콘텐츠 확인
4. 버튼 실행 시 "풀다운"이라고 읽히는 경우

[가상 커서 상태 확인]
- 버튼: "버튼"이라고 읽힘 (정상)
- 풀다운 버튼: "풀다운"이라고 읽힘 (가상 커서 켜진 상태)

[개인 설정 팁]
- Ctrl+Shift+F11로 기능 키 설정 메뉴 열기
- 가상 커서 켜기/끄기를 Alt+1, Alt+2 등으로 변경 가능
- 왼손으로만 조작 가능하게 설정 권장


========================================
10. TTS(텍스트 음성 변환) 관련 단축키
========================================

[Chrome 확장 프로그램 - Read Aloud]
- Alt+P: 페이지 읽기 시작/중지
- 설치 필요: Chrome 웹 스토어에서 "Read Aloud" 검색

[Speechify (유료)]
- Alt+A: 페이지 읽기 시작
- 자연스러운 음성 제공

[Microsoft Edge 기본 TTS]
- Ctrl+Shift+U: Edge 브라우저에서 페이지 읽기
- 별도 설치 불필요
- 200여 가지 목소리 제공
- 추천 목소리: 한국어 "현수"

[일레븐 랩스 (ElevenLabs)]
- Ctrl+Enter: 텍스트 음성 생성
- 유튜브 크리에이터들이 많이 사용
- 유료 서비스


========================================
11. 센스리더 기능 키 설정
========================================

[기능 키 설정 열기]
- Ctrl+Shift+F9

[추천 설정]
- 가상 커서 켜기: Alt+1
- 가상 커서 끄기: Alt+2
- 이유: 왼손으로만 조작 가능, NotebookLM 사용 시 자주 사용

[센스리더 활용의 중요성]
"센스리더의 활용 능력이 곧 AI 활용 능력으로 이어진다"
- AI 서비스 사용 시 웹 브라우저 필수
- 헤딩 이동, 버튼 이동, 가상 커서 제어가 핵심
- 기본기가 탄탄해야 AI 서비스 활용 가능


========================================
부록: NotebookLM 활용 팁
========================================

[소스 제한]
- 무료: 노트당 소스 50개, 노트 1000개
- 유료 (Google Gemini): 소스 300개
- 권장: 소스 10개 이내 (오디오 오버뷰 품질 유지)

[지원 파일 형식]
- PDF, DOC, TXT, 웹페이지, 유튜브 URL
- HWP 미지원 → PDF로 변환 (한글 파일 메뉴 → PDF로 저장 → Ctrl+F)

[활용 사례]
1. 논문 학습: 100페이지 논문을 팟캐스트로 변환
2. 의학 정보: 가족 건강 정보를 오디오로 제작
3. 수업 준비: 교과서 내용을 팟캐스트로 생성
4. 외국 논문: 영어 논문 → 한국어 팟캐스트/보고서

[채팅 탭 vs 스튜디오 탭]
- 채팅 탭에도 오디오/비디오/마인드맵 버튼 있음
- 하지만 오류 발생 가능, 외국어 소스 시 영어로 생성되기도 함
- **권장: 콘텐츠 생성은 스튜디오 탭에서 진행**

작성 후기
========================================

이 문서는 김헌용 강사님의 대전맹학교 디지털 연수 녹취록을 기반으로
